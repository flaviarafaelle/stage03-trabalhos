# Recriando Layout

## 💻 Sobre o desafio
Neste desafio, recriei uma aplicação responsiva, a partir de um layout, para praticar o que aprendi até o **Stage 03** do Explorer da Rocketseat.

Conteúdos abordados:
- Mobile First;
- Unidade de medida flexível;
- Variáveis;
- Animações simples.
